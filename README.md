# ⚜️ Velvet & Gold | Luxury Liquid Gastronomy

Welcome to **Velvet & Gold**, an immersive digital sanctuary designed for the world's most opulent custom beverage experiences. Every aspect of this application is crafted to evoke an atmosphere of refined indulgence, sensory discovery, and modern drink design.

---

## 🎨 Creative & Visual Concept

Velvet & Gold is styled with an eyes-safe, high-contrast, premium dark luxury aesthetic:
- **Liquid Ambience**: A high-definition, color-calibrated background cinematic video loop highlighting rich, warm visual textures.
- **Micro-Interactions**: Fluid, physics-inspired scroll animations that bring the three key signature blends to life as you glide down the page.
- **Custom Fonts**: An elegant typographical pair combining the modern display tracking of *Space Grotesk* with the organic elegance of *Playfair Display*.

---

## 🚀 Key Immersive Features

1. **The Signature Trilogy Showcase**:
   - Dynamic cup landings that float and anchor gracefully via spring-driven scroll transformations.
   - Interactive recipe breakdowns, including botanical compositions, flavor notes, and custom preparation guides.

2. **Bespoke Collage Builder**:
   - A client-side canvas where you can drag, drop, scale, rotate, and adjust the transparency of luxury asset layers.
   - Create your own digital liquid art canvas and export your personal gastronomy print.

3. **Autonomous Flavor Profile Generator**:
   - An intuitive sensory assessment widget to help you define your current drink temperature, key botanicals, and flavor spectrum.
   - Generates a custom virtual brand profile designed for absolute composure.

4. **Embedded Preparation Shaker**:
   - An interactive digital cocktail shaker widget with tactile visual updates.

---

## 🛠️ Technology Stack

- **Framework**: React 18+ with Vite
- **Language**: TypeScript (Type-Safe Architecture)
- **Styling**: Tailwind CSS (Utility-First Responsive Layout)
- **Animations**: Framer Motion (`motion/react`) for spring physics and scroll tracking
- **Icons**: Lucide React

---

## 💻 Local Development

To run the application locally:

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

### 3. Build for Production
```bash
npm run build
```
This prepares static assets inside the `dist/` folder optimized for serverless edge deployment.
